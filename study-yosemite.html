Yosemite Experience - KNBR - KFOG

Objective

This microsite was a customized contest page with a PHP database setup housed on both KNBR.com and KFOG.com.

Challenge

With customized micro-pages we are able to give clients more of a personalized product for them to present to visitors. My challenge was to set up two separate tables in the same database to keep all of the contest entries under one client name. This would make it easier when it came to time to export all of that visitor information to pass along to the client.

Solution

The solution should always involve keeping things simple and clean. Picking colors similar to the images that were used and using a block layout with a high priority in a mobile first approach.

Learned

I learned more about PHP integration and how to set up multiple tables inside the same database while targeting them separately.
