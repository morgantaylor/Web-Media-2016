KNBR Auction Page

KNBR/Giants Community Fund Auction

Objective

Display between 10-25 different packages for the auctions.

Challenge

Finding a way for people to view those auction items in a clean, uncluttered design and is also easily viewed on mobile devices. I wanted people to be able to view one auction package and its details at a time and have those details disappear when you click on a different item.

Solution

One of the drawbacks of using Wordpress is that code will often get stripped out. I had to find a way to work around the system. With CSS transform and transition I was able to simulate what Javascript could have also done. When you click on an item, an accordion drops down to reveal the details of the package. When you click on another package it opens up and the previous one closes. By only showing the necessary information, this keeps the page clean.

Learned

Not being able to use Javascript inside of Wordpress was a challenge. However, after researching how to use CSS3 I was able to simulate an accordion transition with pure CSS, which is really the only thing Wordpress won't strip out.
